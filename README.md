## Scheduler

Running on Windows:

- Development mode: `cd scheduler && run`
- Production mode: `cd scheduler && run prod`

Running on Mac OS X or Linux:

- Development mode: `cd scheduler && ./run.bat` 
- Production mode: `cd scheduler && ./run.bat prod`
